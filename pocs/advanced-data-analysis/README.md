# POC: Advanced Data Analysis & Document Synthesis

## 🎯 Overview
This POC demonstrates how to move beyond basic chatbot interactions by building an automated, data-driven synthesis engine. It leverages the **OpenAI Assistants API** to combine quantitative data (CSV) and qualitative strategy (PDF) into actionable business insights.

## 🧠 The Synthesis Edge: Numbers vs. Strategy
This POC is a **Cross-Format Auditor**. It doesn't just plot charts; it understands the business context:
- **The "Manual" (PDF)**: The AI reads your expansion strategy to learn your goals and benchmarks (e.g., identifying "South Delhi" as the performance standard).
- **The "Books" (CSV)**: The AI then "audits" the raw sales numbers using Python to see if the reality matches the strategy.

**Outcome**: Instead of a generic summary, the AI provides strategic commentary: *"Gurugram sales are failing to meet the South Delhi benchmark defined in your strategy document, likely due to the product cannibalization you warned about."*

## 🚀 Features
- **Complex Multi-File Analysis**: Simultaneously references internal strategy reports and raw sales datasets.
- **Automated Visualization**: Uses the Assistant's native Python environment to generate trend charts and export them as `PNG` files.
- **Stateful Intelligence**: Maintains thread context for iterative "what-if" questioning.
- **NCR Samosa Case Study**: A realistic business scenario focused on regional growth, seasonal trends, and product cannibalization.

## 🛠️ Setup & Usage
1.  **Configure API**: Ensure your root `.env` file has a valid `OPENAI_API_KEY`.
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Generate Test Data**:
    ```bash
    python generate_mock_data.py
    ```
4.  **Run the Analyst**:
    ```bash
    python analysis_engine.py
    ```

## 📂 Project Structure
- `data/`: Contains synthetic sales and strategy files.
- `output/`: Automatically stores charts generated by the AI.
- `analysis_engine.py`: The core orchestration script.
- `generate_mock_data.py`: Creates the complex dataset.
